Atributos

  • autocomplete: permite autocompletar
  • autofocus: al momento de cargar la pag recibe el foco
  • placeholder: texto grisado que sirve como guia
  • name: nombre con el que se identifica al elemento
  • id: idenificador unico
  • required: campo requerido
  • pattern: recibe expresiones regulares
  • multiple: permite subir mas de un archivo
  • disabled: deshabilita el campo
  • readonly: campo de solo lectura
  • value: se le asigna un valor por defecto
